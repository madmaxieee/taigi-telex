## Installation Guide

### Tâi-gí

#### Tē 1 Pōo: Níng-tsong Thêng-sek

1. Tuì ē-té ê liân-kiat kā `TaigiTelex-x.x.x.pkg` lia̍h lo̍h-lâi
2. Phah-khui tú-á lia̍h ê `.pkg` tóng-àn, tsun-tsiàu tsí-sī oân-sîng an-tsong

**Nā-sī khuànn tio̍h "cannot be opened" ah-sī "unidentified developer":**

Tsit-ê su-ji̍p-huat bô iōng Apple khai-huat-tsiá tsìng-su lâi tshiam-sú, tse tuì khai-guân nńg-thé sī tsin tsìng-siông. Í-hā sī kái-kuat ê hong-sik:

1. Khì **System Settings** → **Privacy & Security**
2. Tī **Privacy** lāi-té tiám TaigiTelex pīnn--á ê **Open Anyway**

#### Tē 2 Pōo: Ke-ka Su-ji̍p-huat

1. Khì **System Settings** → **Keyboard** → **Input Sources**
2. Tiám **Edit…** (ah-sī kū pán-pún macOS ê **+**)
3. Tī **Chinese, Traditional** lāi-té suán **Tâi-gí Telex**

> Nā Tâi-gí Telex bô tshut-hiān, tshiánn ting-tshut lí ê Mac liáu-āu tîng ting-ji̍p.

### 繁體中文

#### 步驟 1：安裝套件

1. 從下方連結下載 `TaigiTelex-x.x.x.pkg` 檔案
2. 開啟下載的 `.pkg` 檔案，依照指示完成安裝

**如果看到「無法打開」或「無法識別的開發者」：**

此輸入法未使用 Apple 開發者憑證簽署，這對開源專案來說很正常。請依以下步驟允許安裝：

1. 開啟**系統設定** → **隱私權與安全性**
2. 在**安全性**區段點擊 TaigiTelex 相關訊息旁的**仍要打開**

#### 步驟 2：加入輸入法

1. 前往**系統設定** → **鍵盤** → **輸入來源**
2. 點擊**編輯…**按鈕（或舊版 macOS 的 **+**）
3. 在**繁體中文**下搜尋 **Tâi-gí Telex** 並加入

> 如果 Tâi-gí Telex 沒有出現在輸入法列表中，請登出 Mac 後重新登入。

### English

#### Step 1: Install the Package

1. Download the `TaigiTelex-x.x.x.pkg` file from the link below
2. Open the downloaded `.pkg` and follow the prompts to install

**If you see "cannot be opened" or "unidentified developer":**

This package isn't signed with an Apple Developer certificate, which is normal for open-source projects. To allow it:

1. Open **System Settings** → **Privacy & Security**
2. Click **Open Anyway** next to the TaigiTelex message under **Security**

#### Step 2: Add the Input Method

1. Go to **System Settings** → **Keyboard** → **Input Sources**
2. Click the **Edit…** button (or **+** on older macOS versions)
3. Search for **Tâi-gí Telex** under **Chinese, Traditional** and add it

> If Tâi-gí Telex doesn't appear in the list, log out of your Mac and log back in.
